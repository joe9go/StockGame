Stock Guess

Home Leaderboard

A game to predict stock price for a company and win ponts as you are close to the actual price in the future

The more the user is close to the actual value and the further he is away from the day he's placing th prediction the more points he gets

The website will include a home page(to make predictions), a leaderboard page, login page, and a register page

Used python for the backend and used yfinance to collect current stock price and data and used json package to connect with front end of the website that was created by using Html, Css and Java
